# 教师的信息茧房

思维茧房是指人因长期接触同质化信息、局限于单一视角，逐渐形成封闭的认知圈层，难以接受圈外不同观点的现象。

简单说，思维茧房就是你总看、总听自己喜欢的东西，时间长了就像把自己关在一个“信息小房子”里，看不到外面不一样的想法。

比如有人特别喜欢某类短视频，算法就总推给他类似内容——全是说“某件事就该这样”“某类人都这样”的观点。慢慢他就觉得全世界都跟自己想的一样，等某天听到有人说“其实还有另一种可能”，他会觉得对方错了，甚至根本听不进去。

## 教师的职业生涯路径

你思考过人民教师的职业生涯路径吗？

几乎所有人民教师的职业生涯路径都遵循：大学毕业->考编->成为教师。

大家没发现问题吗？

教师没有踏入过社会，毕业后就进入学校工作，教师的人生是不完整的，他们生活在一个封闭的圈子里，少与社会其他职业有更多交集。其他职业也存在行业和领域的问题，但是我们会经常换工作和搬家，这能让他们接触更多的人。而教师行业几乎都是终身制，即使换工作还是一个体制内。

这造成了教师的信息茧房，他们的思维方式已经脱离现实社会，作出很多匪夷所思的事情，甚至反智行为，这都是正常行为。我举几个例子：

1. 教师是法盲，教师并不比你更懂法律，例如一些老师发现孩子玩手机，于是便没收手机后将其毁坏，他并没有意识到这是犯罪。
2. 体罚和语言侮辱学生这是最常见的，尤其是语言侮辱学生，给学生造成的心理伤害程度无法评估。
3. 泄漏学生隐私与个人信息，例如公开学生的成绩排名、家庭住址、联系方式等敏感信息；私自拆阅学生信件、日记；在网络上传播学生的照片、视频等隐私内容，造成学生名誉损害。
4. 擅自开除学生，非法剥夺学生受教育权，活用软暴力对待学生。表现，无正当理由拒绝适龄儿童入学、强制学生退学、不准学生参加考试或正常教学活动；因学生成绩差、违纪等原因歧视学生，变相剥夺其受教育机会。

不在一一列举，如果把全国案例都罗列出来，可谓罄竹难书。笔者本意不是数落教师罪名，而是反映问题，就是上面问题在教师们的思维茧房内，他们并不认为是违法。

## 教育的初衷已经变了

人们心目中的教师“文化工作者”，知识渊博，身负 “传道、授业、解惑” 之责，然而教师虽然从事文化工作，但是许多教师存在 “职业倦怠”，“躺平”现象，少有教师保持持续学习，已经失去创作能力。如果你的孩子上过网课，你会发现学校的老师与网课老师的差距有多大。

说到“传道、受业、解惑”就更扯蛋了，“教育”在当下社会就是一种“消费”，我管这个叫“学生经济”。

虽然教师受过高等教育，每日从事教学工作，但是行为上与常人并无不同，他们既不不了解读书的意义，也不知道学历的本质是什么。换个角度，我们也不能一味责怪老师，造成如今这种现状更多是社会责任。是国家顶层设计者的责任。

### 学历的本质是什么?

每到高考季，看到无数家长为孩子瞎操心，真让人着急。教育淘汰的不是孩子，是那些对教育认知不足的家长。那些认知不足的家长！往往需要为自己错误的决策付出代价。

尤其是很多生活在社会底层的家庭，家长迷之自信，在平日里还会以关心，我为你好等等自居，表现为热心提供帮助，例如帮别人选车，选房，就医……

这种行为仅仅表现为热心，真的没有任何帮助，他们自己没有汽车，没有房子，却给别人提供建议，并且没有边界感，触及到别人的私人生活，例如恋爱，家庭等等……

当然，他人的建议，不是不能听，前提是他们认知在你之上。就像我前面所说，你千问不要问一个没有谈过恋爱的人怎么脱单。

作为孩子，你更要认清，生活环境会限制你父母的想象力，社会阶层会成为想象力的天花板，如果你生活在底层社会，你的父母无法提供给你任何有价值的信息，甚至他们的价值观是负向的。此时，你需要做的是“孝而不顺”，不要听他们的意见，以及他门身边人的意见。此时你只能靠自己，你可以问问你家境好的同学和他们的父母，你同学的父母提供的信息价值远远大于你父母他们。

之前我谈过教育的本质《教育的目的和意义是什么？》，今天我来谈谈学历的本质是什么？

### 首先谈谈中国社会结构

中国是一个金字塔社会，全产业链工业社会，那么什么是全产业链？

全产业链是指企业从原材料供应、产品研发、生产制造、仓储物流、销售渠道到售后服务等各个环节进行全程把控，形成覆盖产业链上下游的完整产业体系。

美西方呢？西方主要是在高端科技、金融等部分产业链环节占据优势，并非覆盖从原材料供应到终端消费等全产业链各环节，主要依赖全球分工实现资源整合与产业协作。

说白了全产业链包含了，风吹雨打日晒雨淋的漏天工作环境，到24小时吹空调的写字楼环境。从脏活累活，初级原材料加工的汗水经济，到价值链顶端的规则制定。

在这个产业链上，必须每个环节都有人来参与劳动，同时要有充足的劳动力，以应对三班倒的企业生产诉求。

在在《教育的目的和意义是什么？》说过，社会就是一个永不停止的机器，人就是这部机器的零件，要让机器不停运转，就需要不断升级，更换损坏的零件，甚至换新淘汰部分组件。

所谓全产业链，就是从底层劳动开始，折钢筋，扛混凝土，打螺丝，焊接电路板，到踩缝纫机，涂胶水粘橡胶。再到，人工智能，生物工程，卫星上天，光刻机，新材料……

在这个链条上，我们有无限多的岗位，这些岗位也需要无限多的工人、打工人、牛马、社畜……

我们是否庆幸，有这么多岗位？我们是否不发愁找工作了？

现实中并不非如此，由于我们的社会结构，把人分成三六九等（虽然官方不承认，现实就是如此），底层打工人，享受不到社会福利，金字塔顶端的人垄断社会资源。这种状况与满清贵族和大清子民的关系并无差别，我们真的不能怪我们的政府，这是一个终极的哲学问题。即：给底层人高薪，底层人就不会再从事那些底层的工作，这就是所有西方发达国家都存在的问题“养懒汉”，西方发达国家本国国民宁可领救济金，也不会从事社会底层工作。导致他们一遍缺少大量劳动力，一遍花财政养着本国国民，这些懒汉手握选票，谁敢动他们的利益，他就立即让你下台。

最终美西方发达国家的解决方案是开放移民，让移民/非法移民来从事社会底层，白人不愿意干的工作，从上个世纪开始，西方白人就享受了移民红利，他们通过给非法移民希望，让他们心甘情愿为美西方贡献青春。

我们并不是移民国家，我门不可能开放移民，让大量黑人来国内工作，让他们成为二代移民，跟我们的民族生下黑宝宝。所以，全产业链上的每个环节就需要我们自己来完成，工作岗位留给自己人。有了工作岗位，同时还有有人心甘情愿的去做。现实情况是，没人想做这些工作。

但是随着中国经济增长，产业转移，国人也开始飘了，底层人也矫情了，他们也想像西方白人一样，他们甚至还想着，免费医疗，免费教育，等等异想天开事，西方人有的福利，我门要有，西方人没有的福利我门也要，不切实际的想法在底层开始萌芽。这造成了很多低端制造业出现了用工荒，他们不想再让自己的孩子从事低端劳动。

你用屁股想想，国家一定会做出点什么，你懂的！聪明的我们肯定懂的怎么拿捏这些人：）

说到这里你应该明白了吧？我们的教育，是服务于社会机器的，现在社会需要有人下沉到底层制造业，而不是都读大学，然后眼高手低，不想从事底层工作。

教育的本质就谈到这里，下面我分一下社会热点问题，读书无用论。

### 读书无用了，学历贬值了

在我文章《教育的目的和意义是什么？》中已经说的很清楚，“读书”只能掌握一种非体力劳动技能，而改变命运需要用你掌握的非体力劳动技能为社会和企业创造价值。

仍有认知不足的学生和家长，一遍一遍的追问。

学历！只不过是你四年按时出勤给你开具的学费凭证。就如同考驾照，驾照（学历）只能证明你学过驾驶，驾校没有说有了驾照，就会有工作，同时驾校也不包分配。有了驾照，你要自己拿着驾照去应聘司机，才能有收入养活自己。这很难理解吗？

大学里学不到真本事？

我在另一片文章《为什么中国大学培养不出企业需要的人才》中讲的非常清楚。

首先，本科教育是面向学术的，是给后面的学位打基础的，所学的知识篇学术。面向就业，学的一项生存本领，那是专科教育的范畴。

然后，中国教员选拔多是外聘，或者留校晋升，教育们没有在企业干过，无法吧社会主流技术带到学校交给学生。

最后，教员是终身制，他们没有动力更新自己专业的知识，也没有淘汰机制，他们躺平，养老。教学生淘汰的知识。

用上面驾照举例，学校教你的知识“马车设计，维修与保养”，你学到知识，进入社会，根本用不上，因为马车也不存在了。
